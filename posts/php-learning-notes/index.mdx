当双引号中包含变量时，变量会与双引号中的内容连接在一起；
当单引号中包含变量时，变量会被当做字符串输出。

使用Heredoc结构形式的方法来解决该问题，首先使用定界符表示字符串（`<<<`），接着在`<<<`之后提供一个标识符GOD，然后是字符串，最后以提供的这个标识符结束字符串。

```php
<?php 
    $string1 = <<<mark
    Trsdyugcfuihcfyhxseyihvfujvgg
    mark;
    echo $string1;
?>
```

?>

### 类：
```php
class ClassName{
    public  $v1 = “”;
    protect $v2 = “”;
    private $v3 = “”;

    public function Function1(){
        return($v1.$this->v3);
    }
    static function Function2(){
        return($v2);
    }
}

$v4  = “ClassName”;
$obj = new ClassName();
$obj = new $v4();
echo $obj -> v1;
echo &obj -> Function1();
echo $obj :: Function2();
```

静态方法中，$this伪变量不允许使用。可以使用self，parent

属性的重载通过__set，__get，__isset，__unset来分别实现对不存在属性的赋值、读取、判断属性是否设置、销毁属性。\
方法的重载通过__call来实现，当调用不存在的方法的时候，将会转为参数调用__call方法，当调用不存在的静态方法时会使用__callStatic重载。\
通过serialize方法将对象序列化为字符串.\
通过unserialize将字符串反序列化成对象.
