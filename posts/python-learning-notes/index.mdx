### 『List a = [,,,,] 可修改』
 - ```a[n]```正序;```a[-n]```倒序
 - ```a.append()```总是把新的元素添加到list的尾部。
 - ```a.insert(n, '')```
 - ```a.pop([n])```默认最后一个，剔除掉并打印
 - ```a[ : : ]```划片分割，起始:结束:间隔 第一个为“”表示从0，前两个为“”表示全集
 - ```[ ??? for ? in range(,,)]```
### 『Tuple a = (,,,,) 不可修改 』
 - ```a[n]```正序;```a[-n]```倒序
 - ```a = ('', )```单元素定义
 - ```a = ([''], )```内嵌list
