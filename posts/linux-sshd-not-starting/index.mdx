### linux下无法启动sshd服务
```bash
## redhat，fedora，centos：
sudo yum install sshd 或
sudo yum install openssh-server

## debian，ubuntu，linux mint：
sudo apt-get install sshd 或
sudo apt-get install openssh-server
```
